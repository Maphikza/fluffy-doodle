Welcome to The Legislative Library

We created this website to make it easier to read and understand legislation. On this site, you'll find actual legislation as well as simplified summaries in plain language for each section. We hope you find this site helpful and easy to use!

We created this website to make it easier to read and understand legislation. In order to understand things better it sometimes helps to read a summary. So, we have added a plain language summary to each section. We hope that this will help you be able to more easily and quickly digest the information and have a good start for communicating it to others. All you have to do is click on the legislation button, select the legislation you are looking for, and you will be directed to the page with the piece of legislation you selected.

Please note: the summary is not meant to replace the law; the laws must always take precedence.
